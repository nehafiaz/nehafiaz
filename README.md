## Engineering Philosophy

- **Clean architecture** — clear boundaries between UI, state, and data so nothing turns into spaghetti under pressure
- **Security by default** — validate inputs, lock down data with RLS, never trust the client
- **Performance with intent** — optimistic updates and lean models, but only where they earn their complexity
- **Design as engineering** — motion, spacing, and hierarchy are part of the build, not an afterthought
- **Honesty over hype** — every project on this profile is real, shipped, and can be demoed
