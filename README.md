## 🚀 Projects

### 🌿 [GreenSense](https://github.com/nehafiaz/greensense) — plant disease detection from leaf images

Real-time computer vision diagnosis for growers who don't have an expert on site.

- Fine-tuned MobileNetV2 with two-phase training and data augmentation — **94.76% test accuracy** <!-- add: on [dataset name], N classes -->
- 5-layer image validation pipeline before inference (file type, size, content checks)
- Flask API hardened and optimized for production inference

`TensorFlow` `MobileNetV2` `Flask` `Python` `OpenCV`

### 📚 [Words Versus](https://github.com/nehafiaz/words-versus) — gamified vocabulary app with real-time sync

Fast word-pair review across 750+ curated pairs in 20 categories.

- Three-layer state architecture: Zustand (UI state) + TanStack Query (server cache) + Supabase (persistence)
- Optimistic updates for instant feedback, with reliable background sync and rollback on failure

`React` `TypeScript` `Zustand` `TanStack Query` `Supabase`

### 🩺 [MediDesk](https://github.com/nehafiaz/medidesk) — practice management for healthcare providers

Mobile-first scheduling, records, and operations built around real clinical workflows.

- Supabase Row-Level Security designed in from day one — zero trust in the client
- Full auth flows with Expo Router, real-time sync across devices

`React Native` `Expo` `Supabase` `PostgreSQL`

### 💼 [nehafiaz.com](https://nehafiaz.com) — portfolio

Dark-mode-first personal site built on real, verifiable project data. `Next.js` `TypeScript` `Tailwind` `Vercel`
